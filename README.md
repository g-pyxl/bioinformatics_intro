# Bioinformatics Intro Exercise

Welcome! This is a short exercise to help us assess your familiarity with basic development tools.

## Task

1. **Fork this repository** on GitHub (use the "Fork" button in the top-right corner).
2. **Open your fork in a Codespace** by navigating to `https://codespaces.new/<your-username>/bioinformatics_intro` (or click the green "Code" button on your fork and select "Create codespace on main").
3. **Open a terminal** in the Codespace (`Terminal > New Terminal`).
4. **Run the Python script**:
   ```bash
   python hello.py
   ```
5. You should see `Hello` printed in the terminal.
